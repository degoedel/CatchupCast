Podcatchup

Podcast manager designed for streaming and marking episodes as read

Download this project as a .zip file Download this project as a tar.gz file

Welcome to PodCatchup HomePage.

This software is intended to Manage and stream podcast episodes on a desktop. It is intended for Windows. What you can do with it :Â  Add or remove a podcast through its feed. Stream an episode. Mark it as read or new. If you pause it, or quit the app, you will be able to resume the stream where you left it. And that's all.

You can download the installer for the current version here .

Requirements

  • Windows
  • .Net 4.5 is required. So you should be fine on Windows 7 and over as long as you're up to date. Since the application only does streaming, you need an internet connection obviously.

Dependencies

Outside of Prism, this project also uses:

  • LiteDB by @mbdavid for data persistence
  • CSCore by @filoe for audio streaming
  • Mahapps by @MahApps for theming
